Introduction

This site summarizes the complexities in terms of space and time (Big O) of the most important algorithms and operations in common data structures. It is 100% open-source, with no ads and has a friendly printable version. If you have any comment please create an issue on Github. To know more about the project and see the credits take a look at the readme file.
